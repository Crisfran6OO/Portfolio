# 🚀 Portafolio Profesional & Servicios Digitales

Un portafolio web moderno, responsivo y dinámico construido con **React + TypeScript + Vite**. 
Este proyecto está diseñado con un enfoque de "doble entrada", permitiendo a los visitantes explorar tanto la faceta de desarrollador profesional (orientada a reclutadores), como los servicios digitales orientados a potenciar negocios.

## 🌟 Características Principales

* **Doble Ruta de Navegación:**
  * **Página Profesional (`/profesional`):** Orientado a reclutadores y empresas. Destaca la experiencia, stack tecnológico y foco en el desarrollo de software.
  * **Página de Servicios (`/servicios`):** Orientado a clientes directos que buscan digitalizar su negocio con páginas web, e-commerce y sistemas a medida.
* **Diseño Glassmorphism:** Interfaz inmersiva estilo *dark-glass*, moderna y pulida utilizando capas semi-transparentes (blur) y gradientes llamativos.
* **Animaciones Fluidas:** Transiciones de página sin cortes gracias a `framer-motion` y efectos de aparición mientras se hace scroll en la página.
* **100% Responsivo:** Interfaz adaptada con el patrón *mobile-first* para brindar la mejor experiencia en celulares, tablets y monitores grandes.

## 🛠️ Tecnologías Empleadas

El proyecto está diseñado bajo una estructura modular, utilizando las herramientas más rápidas del ecosistema frontend actual:

* **Librería UI:** [React 19](https://react.dev/)
* **Lenguaje:** [TypeScript](https://www.typescriptlang.org/)
* **Estilos:** [Tailwind CSS v4](https://tailwindcss.com/)
* **Enrutamiento:** [React Router v7](https://reactrouter.com/)
* **Animaciones:** [Framer Motion](https://www.framer.com/motion/)
* **Build Tool:** [Vite](https://vitejs.dev/)

## 📂 Visión General de la Estructura

```text
src/
├── components/   # Componentes de UI reutilizables (Navbar, Botones, SectionHeader)
├── data/         # Información de perfil estática (personal.ts, services.tsx)
├── hooks/        # Custom hooks locales (como intersección de scroll)
├── layouts/      # Envolturas base para estructurar las páginas (MainLayout)
├── pages/        # Vistas principales separadas por audiencias
├── types/        # Modelos e interfaces de TyeScript
├── index.css     # Estilos base y tokens de diseño personalizados 
└── App.tsx       # Configuración global del enrutador
```
